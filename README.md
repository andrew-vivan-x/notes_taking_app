# Notes Taking App

A full-stack Notes Taking Application where users can **sign up, log in, create, edit, view, and delete notes** securely.  
This project is built as part of the **I2Global Full Stack Developer Assignment** using the required technologies.

---

## Tech Stack Used

| Layer | Technology Used |
|-------|-----------------|
| **Frontend** | Next.js, Tailwind CSS, Redux Toolkit, Axios, Framer Motion |
| **Backend** | FastAPI, SQLAlchemy, JWT Authentication, PyMySQL |
| **Database** | MySQL |
| **Containerization** | Docker & Docker Compose |

---

## Features

- JWT-based authentication system (Signup / Login)
- CRUD operations for notes (Create, Read, Update, Delete)
- Responsive UI using Tailwind CSS
- State management using Redux Toolkit
- Optimized API structure (user + notes modules)
- Docker containerization (Frontend + Backend + Database)

---

## Project Setup & Installation

### Clone the repository
```sh
git pull https://github.com/andrew-vivan-x/notes_taking_app.git
```

### Run using Docker
```sh
docker compose up --build
```
